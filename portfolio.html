3. UX

This theme introduced me to the fundamentals of UX (User Experience) and UI (User Interface) design, focusing on the interaction between users and digital interfaces.

Topic website

For the main project, I designed a topic-based website using user-centered design principles. The process began with brainstorming to define the purpose, audience, and user stories. From there, we conducted desk research to understand user needs. We then created moodboards, styletiles, and user flows to guide the site's structure and visual design. These steps ensured the design aligned with the intended user experience.

Prototyping included usability tests like 5-seconds and think-aloud tests, leading to refined layouts for coding. After completing the website, we performed a Lighthouse test to optimize its performance. This project provided valuable experience with the design process, usability testing, and presenting insights and outcomes to an audience.

4. Animation

Game website

In this theme, I learned JavaScript and CSS animations to create the "Splash Sprint" game. We began by developing prototypes to visualize the game's flow and interactions. Activity and state machine diagrams were used to map out the interactive flow and structure the game's logic. Once the planning phase was complete, I started designing the game graphics in Adobe Illustrator.

Afterward, I moved on to coding the game elements, focusing on collecting "good" items and avoiding "bad" ones, which affected points and lives. JavaScript was used for the game logic, and CSS for animations. Finally, we tested the game with our classmates.

This project helped me enhance my skills in frontend technologies, interactive design, sound integration, and creating dynamic user experiences.

5. Content

Passion website

For this passion project, I created a website showcasing the interests of someone deeply passionate about their craft. We chose to feature my roomie, who is currently an apprentice goldsmith. While the planning phase, including filming and photography, was done in groups, the editing and website creation were carried out individually.

This project also introduced me to animated vector graphics (LottieFiles), which I created using Adobe After Effects. I learned about the advantages of LottieFiles, such as their lightweight nature and smooth integration into websites. The video content was produced with Adobe Premiere Pro and Audition, where I combined the footage with photography to effectively showcase the passion behind the craft.

Business website

I worked in a group to redesign a website for the company Vaerkstedet, a combination of a jewelry shop and workshop in Copenhagen. The focus was on optimizing functionality and design, ensuring that the website followed the shop's identity. The process gave me valuable experience in project organization, team collaboration, and applying a structured design approach in a real-world context. Furthermore, this project served as groundwork for our following exam.
