4. Animation

In this theme, I learned JavaScript and CSS animations to create the Splash Sprint game. I designed and coded game elements, focusing on collecting "good" items and avoiding "bad" ones, earning points and losing lives. The project involved pre-production planning, shooting with a smartphone and external microphone, and editing the video. I also applied techniques for creating animated graphics and optimized the layout using the skills I learned from previous themes.

This project allowed me to explore content creation, organization, and video editing, helping me communicate a subject's passion effectively.
